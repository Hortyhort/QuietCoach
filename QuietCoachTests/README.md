# QuietCoachTests

Unit tests for QuietCoach.

## Setup

To enable these tests:

1. Open QuietCoach.xcodeproj in Xcode
2. File > New > Target
3. Select "Unit Testing Bundle"
4. Name: QuietCoachTests
5. Add the test files to the new target

## Test Files

- **FeedbackEngineTests.swift** - Tests for FeedbackEngine scoring logic
- **SpeechAnalysisTests.swift** - Tests for speech analysis components

## Running Tests

Press Cmd+U in Xcode or use `xcodebuild test`.
